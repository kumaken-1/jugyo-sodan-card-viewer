# 授業相談カード大全 Web版ビューア

授業相談カード大全のカード画像とTSVデータを閲覧するための、静的Webアプリです。

このアプリはAI相談アプリではありません。カード文言を生成・改善・要約せず、TSVにある値をそのまま表示します。

## 公開URL

GitHub Pagesで公開後、次の形式のURLから利用できます。

```text
https://kumaken-1.github.io/jugyo-sodan-card-viewer/
```

## 主な機能

- トップ画面の3×3マンダラート表示
- カテゴリ画面の3×3カード表示
- カード詳細表示
- カード検索
- 検索補助タグ
- 気になるカードの一時保存
- 今日見る一点メモ
- メモ欄
- 使い方ガイド
- 画像がない場合の「準備中」表示

## 使い方

1. トップ画面からカテゴリ画像を選びます。
2. カテゴリ画面で見たいカード画像を選びます。
3. 詳細画面でカード画像とTSV情報を確認します。
4. 検索欄やタグから、気になるカードを探せます。
5. 気になるカードは最大5枚まで一時保存できます。
6. 必要な文はメモ欄にコピーして貼り付けられます。

## データについて

アプリで使用するカード情報は、次のTSVをもとに表示しています。

```text
assets/data/app_cards_full_values.tsv
```

検索補助タグは、カード本文とは分けて次のファイルで管理しています。

```text
assets/data/card_tags.json
assets/js/card_tags.js
```

検索補助タグは、カードを探しやすくするための補助メタデータです。カード本文そのものではありません。

## プライバシーと保存

- 個人情報は含めていません。
- 外部APIは使用していません。
- ログイン機能はありません。
- 利用履歴は保存しません。
- 気になるカードとメモ欄は `sessionStorage` に一時保存します。
- ブラウザ画面を閉じると、一時保存した内容は消えます。

## 技術構成

- HTML
- CSS
- JavaScript
- TSV
- JSON
- JPEG画像

外部ライブラリや外部APIは使用していません。

## ローカルで使う場合

このフォルダをそのままダウンロードし、`index.html` をブラウザで開いてください。

```text
index.html
assets/
```

`index.html` だけを移動すると、画像やデータが読み込めません。必ずフォルダ構成を保ったまま使ってください。
